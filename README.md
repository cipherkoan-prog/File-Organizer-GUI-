# 📁 File Organizer (Python + Tkinter)

## 📌 Description

This is a simple File Organizer application built using Python and Tkinter.
It helps users automatically organize files in a selected folder into different categories like Images, Videos, Documents, Music, etc.

## 🚀 Features

* Select any folder using a GUI
* Automatically sorts files by extension
* Creates folders like Images, Videos, Documents, etc.
* Moves files into their respective folders
* Easy-to-use interface

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* OS Module
* Shutil Module

## ▶️ How to Use

1. Run the Python script
2. Click on **"Select Folder"**
3. Choose the folder you want to organize
4. Click **"Organize Folder"**
5. Done! Your files will be sorted automatically

## 📂 Supported File Types

* Images: jpg, png, jpeg, gif
* Videos: mp4
* Documents: pdf, docx, txt
* Music: mp3
* Archives: zip
* Programs: exe

## ⚙️ Requirements

Make sure Python is installed (Python 3.x)
